# Applitrack
This is kind of an ERP for job search.
Still in progress. User tracks every vacancy and response they apply to and receive, and the app gives insights and meaningful data to optimize processes like a company would. 

It's meant to store data both on local storage and on a server. When the frontend is produced I might work on the backend. 

This insights are meant to answer questions like:
Is this a right time for a job change for me?
What salary range could I be earning at this point of my career?
What skills would better fit my career path at the current job market?
Should I expand horizontally or vertically?
